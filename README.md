# MAPALA-NEWS — Portal Berita MAPALA Universitas Semarang

Media Informasi & Publikasi Resmi MAPALA Universitas Semarang.

---

## Deskripsi

Website publikasi berita profesional satu halaman untuk kegiatan **Konservasi TBRS MAPALA USM**, menampilkan artikel lengkap, galeri dokumentasi, lightbox foto, dan desain responsif yang terinspirasi dari portal berita nasional.

---

## Struktur Folder

```
berita/
├── index.html          ← Halaman utama
├── css/
│   └── style.css       ← Seluruh styling (Vanilla CSS)
├── js/
│   └── script.js       ← Seluruh interaktivitas (Vanilla JS)
├── assets/
│   ├── logo-usm.png    ← Logo Universitas Semarang
│   ├── logo-mapala.png ← Logo MAPALA USM
│   ├── hero.jpg        ← Foto grup outdoor (banner Super Adventure) — HERO
│   ├── talkshow.jpg    ← Foto sesi talkshow lingkungan
│   ├── aula.jpg        ← Foto bersama narasumber di aula
│   ├── buka-puasa.jpg  ← Foto buka puasa bersama
│   ├── konservasi1.jpg ← Foto aksi bersih lingkungan
│   ├── konservasi2.jpg ← Foto aksi penanaman pohon
│   ├── gallery1.jpg    ← Foto galeri 1
│   ├── gallery2.jpg    ← Foto galeri 2
│   └── gallery3.jpg    ← Foto galeri 3
├── robots.txt
├── sitemap.xml
└── README.md
```

## Cara Menjalankan

Cukup buka file `index.html` di browser, atau gunakan ekstensi **Live Server** di Visual Studio Code.

---

## Deploy ke GitHub Pages

1. Upload seluruh isi folder ke repository GitHub
2. Buka **Settings → Pages**
3. Pilih branch `main`, folder `/ (root)`
4. Klik **Save**
5. Website akan tersedia di `https://[username].github.io/[repo-name]/`
